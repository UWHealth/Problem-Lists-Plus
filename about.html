About Problem List MD

This web site demonstrates proof-of-concept that useful problem list metadata can be based on expert clinician consensus. The content experts are located at academic or large community hospitals. The experts have donated their time, and the metadata is intended to be available as open source content. We are seeking funding to continue the curation of content development.

The Problem List MD site will provide metadata for problems for use with aggregate display tools and with clinical decision support (CDS). The metadata content uses publically available vocabularies such as SNOMED, Rx Norm and LOINC. The content is curated at the UW Health, associated with the University of Wisconsin.

Problem Concept Maps are being developed to drive aggregate data displays from the problem list. The first maps include lab and medications that are relevant to a particular problem. Later maps will include imaging studies, procedures and documents that should be included in a problem based aggregate display.

Reconciliation/Cleanup Categories are being created for common problems. Vendors can create software that utilizes these category metadata to drive CDS tools for managing the problem list.
